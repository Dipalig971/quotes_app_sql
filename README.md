# quotes_app_sql

A simple Flutter application for managing and displaying quotes. The application uses the sqflite package for local database management.

## Features

- View all quotes
- Edit existing quotes
- Delete quotes


## Dependencies
```
flutter: SDK
sqflite: ^2.0.0+4
path_provider: ^2.0.1

```
